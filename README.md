# Named-Entity-Recognition-using-Bidirectional-LSTM

> [Named-entity recognition](https://en.wikipedia.org/wiki/Named-entity_recognition) (NER) (also known as entity identification, entity chunking, and entity extraction) is a sub-task of information extraction that seeks to locate and classify named entities mentioned in unstructured text into pre-defined categories such as person names, organizations, locations, medical codes, time expressions, quantities, monetary values, percentages, etc.
